Adidas Girls Website

2024 | Role: UX/UI Designer, Front-End Development

Concept

Adidas Girls is a specialized e-commerce website exclusively designed for women’s sportswear and equipment. The platform focuses on empowering active women by providing a curated shopping experience that blends high-quality performance gear with modern style.

Technical Implementation

  • UI/UX Design: Designed the website's interface and user experience using Figma.
  • Front-end Development: Built the website structure and responsive styling using HTML5 and CSS.
Note: This version serves as a functional prototype. Some features are still under development due to project time constraints.
