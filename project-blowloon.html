Blowloon IOT Project

2025 | Role: UX/UI Designer, Developer

Concept

Blowloon is a vibrant, fast-paced game for two players set in a playful and imaginative world. Players choose their side—Team Rainbow or Team Clouds—and compete in an exciting balloon-blowing battle.

Technical Implementation

  • UI/UX Design: Designed the user interface using Figma.
  • Game Engine: Built the core game mechanics and environment in Unity.
  • Programming: Developed the game logic, scoring system, and hardware communication using C#.
  • Hardware Integration: Implemented Arduino with Air Pressure Sensors (Blow Sensors) as the primary controller, allowing players to physically blow to inflate the in-game balloons.

Contributions

Contributed to the game interface (UX/UI) design in Figma, implemented elements in Unity, developed game logic using C#, and integrated with a physical blowing sensor via Arduino IDE.
