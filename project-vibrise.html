Vibrise Website

2024 | Role: UX/UI Designer, Front-End Development

Concept

Vibrise is a music streaming platform concept that goes beyond just listening. It serves as a creative space for everyone to create, share, and promote their own musical works, fostering a community of artists and listeners.

Technical Implementation

  • Conceptual Design: Defined the target audience and platform features to support both listeners and creators.
  • UI/UX Design: Designed a user-friendly interface using Figma, focusing on music discovery and artist promotion.
  • Front-end Development: Built the website structure and styling using HTML5 and CSS3.
  • Interactivity: Implemented JavaScript for responsive elements and user interactions.

Contributions

Designed website UX/UI using Figma and developed the Front-end using HTML, CSS, and JavaScript, including the conceptual design and defining the target audience for the website.

Note: This version serves as a functional prototype. Some features are still under development due to project time constraints.
