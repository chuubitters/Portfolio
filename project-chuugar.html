Chuugar Website

2025 | Role: UX/UI Designer, Front-End Development

Concept

CHUUGAR is a beauty community platform dedicated to authentic user reviews, expert makeup tips, and the latest trends. It’s a space where beauty lovers can discover the right products, share experiences, and exchange creative ideas with a like-minded community.

Technical Implementation

  • Design & Strategy: Conducted conceptual design and defined the target audience, including the creation of User Personas and Site Maps.
  • UI/UX Design: Designed the website's interface and user experience using Figma.
  • Front-end Development: Developed the client-side application using Vue.js and JavaScript.
  • Back-end Development: Implemented the server-side logic and API using PHP and the Laravel framework.
  • Database Management: Designed and managed the relational database system using MySQL.
  • DevOps & Environment: Utilized Docker for containerization and consistent environment setup.

Contributions

Designed website UX/UI using Figma, developed the Front-end with Vue.js/JavaScript, implemented the Back-end with PHP/Laravel, managed the database using MySQL, and utilized Docker for environment setup, including the conceptual design and defining the target audience for the website.

Tools Used

Figma Vue.js Laravel PHP MySQL Docker VSCode
